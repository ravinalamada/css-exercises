Dimension

The default value for these properties are:

  • height:auto
  • width:auto
  • min-width:0
  • max-width:none
  • min-height:0
  • max-height:none

I think these default value will not change unless you set the width height.

If the min-width is greater than width, the min-width will be applied.

If the max-width is less than width, the width will be applied.

If the min-width is greater than max-width, the min-width will be applied.

The content will overflows out of the cell.

The height of the container will depend on height of the container.

The max-width will be implied.
