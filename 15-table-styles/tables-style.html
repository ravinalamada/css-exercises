Table styling

The default value of those properties

  1. table-layout: auto
  2. border-spacing: collapse
  3. border-collapse: 2px
  4. caption-side: top
  5. empty-cells: show

The html elements which correspond to these values

  1. table: <table>
  2. table-row-group: <trgroup>
  3. table-header-group: <thgroup>
  4. table-footer-group: <tfgroup>
  5. table-row: <tr>
  6. table-cell: <td>
  7. table-colunm-group: <colgroup>
  8. table-caption: <caption>
