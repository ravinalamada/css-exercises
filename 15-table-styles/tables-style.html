Table styling

The default value of those properties

  • table-layout: auto
  • border-spacing: collapse
  • border-collapse: 2px
  • caption-side: top
  • empty-cells: show

The html elements which correspond to these values

  • table: <table>
  • table-row-group: <trgroup>
  • table-header-group: <thgroup>
  • table-footer-group: <tfgroup>
  • table-row: <tr>
  • table-cell: <td>
  • table-colunm-group: <colgroup>
  • table-caption: <caption>
