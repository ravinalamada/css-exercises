Table styling

These are the default value of those properties:

  • table-layout: auto
  • border-spacing: 2px
  • border-collapse: separate
  • caption-side: top
  • empty-cells: show

The html elements which correspond to these values

  • table: <table>
  • table-row-group: <tboby>
  • table-header-group: <thead>
  • table-footer-group: <tfoot>
  • table-row: <tr>
  • table-cell: <td>
  • table-colunm-group: <colgroup>
  • table-caption: <col>

The downsides of using the various `table*` values for `display` on a set of elements, thereby simulating a table layout on non-table elements is that you will be missed the related tabular.

Table-layout:auto depends on how wide the content is, whereas within the table-layout:fixed distributes the width of the cell into 50% and 50%.

Yes, we can use border-spacing, but all those no-table elements have to be set its own spesific display such as display:table-row,table-cell.
