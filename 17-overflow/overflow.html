About overflow property and clipping

Overflow

If the witdh restraint is placed on a box, the content and text will be contained in that box neatly.

The default values of these properties:

  1. overflow:visible
  2. clip: auto
  3. clip-path: none
  4. text-overflow: ellipsis

The reason why we use these properties is that we want the content or the text to fit in a box neatly. They are useful in dealing with the overflowing content for shake of the content fit inside a box.

The browser let the content overlape to text below.

The Ellipsis (...) is key word which is used when you don't want to write the content fully.

The text-overflow requires tow properties with its specific value which are white-space:nowrap, overflow:hidden and display:block or inline-block.
