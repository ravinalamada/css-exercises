What is a box model?

Box model styles the pages with different boxes, and it occupies their fonction differently. It is useful for building web pages because it makes your page has a good layout. There are 2 different box model which you can use - standard box and an alternate box model.It is better to use an alternative model because it introduced some time after the standard box model. According to this research (developper mozilla.org, Box model-learn developper,MDN,jan 15 2020,11/03/2020).
